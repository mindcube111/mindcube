## 📝 第一步：准备 Git 仓库

### 选项 A：使用 Git（推荐）

#### 1. 检查是否有 Git 仓库

在项目目录运行：
```bash
git status
```

#### 2. 如果没有，初始化 Git

```bash
git init
git add .
git commit -m "Initial commit with Cloudflare Functions backend"
```

#### 3. 创建 GitHub 仓库

- 访问：https://github.com/new
- 创建新仓库

#### 4. 推送代码

```bash
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

---

### 选项 B：跳过 Git（快速测试）

如果你只是想快速测试，可以先：
1. 构建项目：`npm run build`
2. 在 Cloudflare Pages 中选择 "Upload assets"
3. 上传 `dist` 目录

---

## 🌐 第二步：在 Cloudflare 创建 Pages 项目

### 1. 进入 Pages

在 Cloudflare Dashboard：
- 左侧菜单 → **"计算和 AI"** → **"Workers 和 Pages"** → **"Pages"**

### 2. 创建项目

- 点击 **"Create a project"**
- 选择 **"Connect to Git"**
- 授权并选择你的仓库

### 3. 配置项目

```
Project name: psychological-assessment-platform
Production branch: main
Framework preset: Vite (或 None)
Build command: npm run build
Build output directory: dist
Root directory: /
```

### 4. 保存并部署

- 点击 **"Save and Deploy"**
- 等待部署完成（3-5 分钟）

---

## 🔗 第三步：绑定 KV

### 1. 进入项目设置

- 点击你的 Pages 项目名称
- 点击 **"Settings"** 标签

### 2. 进入 Functions

- 左侧菜单点击 **"Functions"**

### 3. 添加 KV 绑定

- 找到 **"KV namespace bindings"**
- 点击 **"+ Add binding"**
- 填写：
  ```
  Variable name: DB
  KV namespace: mindcube
  ```
- 保存

---

## 🔐 第四步：配置环境变量（可选）

### 在 Settings > Environment Variables

- 添加 `JWT_SECRET`（生成随机字符串，至少32字符）

---

## ✅ 完成！

部署完成后，你会得到一个 URL，例如：
```
https://psychological-assessment-platform.pages.dev
```

访问这个 URL 就可以看到你的网站了！
